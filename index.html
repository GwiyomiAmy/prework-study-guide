HTML

  • The head element contains information about the webpage.
  • The meta element is contained within the head element and defines metadata such as character set, description, key words, author, and viewpoint.
  • The body element represents the visible content shown to the user.
  • The h1-h6 elements are different levels of headings for a text block.
  • The ul, ol, and li elements are unordered lists (bullet points), ordered lists (numbered points), and list items.
  • The img element contains information about images that are displayed on a webpage; src being the location of the image and alt giving a description of the image.
  • The br element creates a break between two blocks of content.
  • The a element creates a webpage link, and href points to the URL.

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicated how much space we want around the content inside an element
  • DRY: use a comma to include two selectors for the same CSS rule.
  • text-shadow: 2px 2px 5px #; is an example of how to add a shadow to text.
  • font-family: Verdana, Arial, sans-serif; is an example of how to choose a font for a page. Always have back up fonts in case a browser cannot display desired font and always end with a generic font.
  • border-radius: 25px; is an example of how to add curved edges to a block.
  • If you want to make your box shadow blurry, add a third px after the first two.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b feature/branch name: creates a new branch and switches to it.
  • git add -A: adds all changes.
  • git commit -m "description": creates commit.
  • git pull origin main: received branch modification into local environment.
  • git push origin branch name: pushes local changes to GitHub branch.
  • git checkout main: switches from branch back to main.

JavaScript

  • Control flow is the oder in which a computer executes code in a script.
  • A const tells what remains constant.
  • A variable is a named container that allows us to store data in our code.
  • A function gives instructions for what the code will do.
  • A console.log will add content to the console.
  • A conditional statement will only execute if the statement is true.
  • An array is a single variable that is used to hold a group of related data.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
